# LocalWrite Assistant 安装指南

## 安装步骤

1. 解压下载的 ZIP 文件到您的计算机上
2. 打开 Chrome 浏览器，在地址栏输入 `chrome://extensions/`
3. 在右上角启用"开发者模式"（Developer mode）
4. 点击"加载已解压的扩展程序"（Load unpacked）按钮
5. 选择您解压的 LocalWrite Assistant 文件夹（包含 manifest.json 的文件夹）
6. 扩展程序将被安装并显示在您的扩展列表中

## 配置 Moonshot API

安装完成后，您需要配置 Moonshot API 才能使用语法纠错功能：

1. 点击浏览器右上角的扩展图标，然后点击 LocalWrite Assistant
2. 在弹出的设置面板中，选择"模型设置"选项卡
3. 选择"自定义 API"作为模型提供商
4. 填写以下信息：
   - **API URL**：`https://api.moonshot.cn`
   - **API 密钥**：`sk-mbGQSzEuCZKbvhpvyp4Ke6XawQmVSJx0U3kA9kbPQT8LCSMG`
   - **模型名称**：`moonshot-v1-8k`
   - **温度**：保持默认值 0.3 或根据需要调整

5. 点击"保存设置"按钮
6. 点击"测试连接"按钮确认连接成功

## 使用方法

配置完成后，您可以在任何网站上使用语法纠错功能：

1. 打开任何包含文本输入框的网站（如 Gmail、Twitter 或 Google Docs）
2. 在输入框中输入包含语法错误的文本，例如：
   - 主谓不一致错误：`I has a new book`（正确应为：`I have a new book`）
   - 冠词使用错误：`I bought a apple`（正确应为：`I bought an apple`）
   - 逗号拼接错误：`I went to the store, I bought milk`（正确应为：`I went to the store, and I bought milk`）
3. 扩展程序将自动检测错误并用红色波浪线标注
4. 点击建议面板中的"一键替换"按钮应用修正

## 快捷键

- `Ctrl+Enter`：应用顶部建议
- `Ctrl+Shift+Space`：切换建议面板显示/隐藏

## 注意事项

- 所有文本处理都通过 Moonshot API 完成，请确保您的网络连接正常
- 如果 API 连接失败，扩展程序将回退到基本的规则匹配模式进行语法检查
- 如果遇到任何问题，请尝试清除缓存、刷新页面或重新启动浏览器
